# インフラ維持費用ランキング（AWS無料枠終了後）

## Q
本番環境のAWSリソースで、無料枠終了後に維持費用が高いものは何か？

## A
少数ユーザー前提で、RDSとEC2が全体の6〜7割を占め最も高額。合計は月額$33〜44程度。

## 費用ランキング

| 順位 | リソース | 構成 | 概算月額(USD) | 備考 |
|------|---------|------|------------:|------|
| 1 | RDS (PostgreSQL) | db.t3.micro, Single-AZ, 20GB gp3 | $15〜18 | インスタンス時間課金。最も高額 |
| 2 | EC2 (ECS on EC2) | t3.micro, 1台常時稼働 | $8〜10 | 750h/月の無料枠喪失 |
| 3 | Cognito | ユーザープール (MAU課金) | $5〜6 | 標準利用＋少人数なら50,000 MAU無料枠で実質無料の可能性あり。高度なセキュリティ機能有効時は高額化 |
| 4 | CloudFront + S3 | SPA配信 | $1〜3 | ストレージ + リクエスト + 転送量 |
| 5 | Secrets Manager | シークレット数個 | $1〜2 | $0.40/シークレット/月 + API呼び出し |
| 6 | Lambda | リマインダー通知用 (Node.js) | $0.5〜1 | 月100万リクエスト無料枠あり |
| 7 | DynamoDB | FCMデバイストークン管理 | $0.5〜1 | 少量ならプロビジョンド25GB無料枠内 |
| 8 | EventBridge | Lambda トリガー | 〜$0.5 | スケジュールルールはほぼ無料 |
| 9 | CloudWatch | Logs / Metrics / Alarms | $0.5〜2 | ログ取り込み量・保持期間による |
| 10 | SQS / SNS / SES | 通知連携・メール送信 | 〜$0.5 | 少量ならほぼ無料 |

## 補足
- 合計概算: 約$33〜44/月（約5,000〜6,500円/月）
- VPS移行でRDS・EC2をDocker化すればVPS1台（月$5〜10）に集約可能
- Cloudflare（Tunnel / WAF / Workers）は無料プラン範囲のため計上外
- OpenAI API / Qdrant(VPS) はドキュメント方針に従い計上外
