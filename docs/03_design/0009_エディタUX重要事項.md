# エディタUX重要事項

## 重要度

**極めて重要** - 非エンジニアユーザーのUXを保つために必須

---

## Q: 装飾子非表示時にバックスペースで削除すると表示が崩れる？

### A: 自前実装で対応可能

- `**太字**` を非表示にしている状態でバックスペースすると、意図しない部分が消える可能性
- CodeMirror 6のトランザクションフィルタで修飾子ペアを検出し整合性を保つ
- Backspaceキーをカスタムコマンドでオーバーライド
- Zennの実装が参考になる

---

## Q: 日本語入力時に変換途中のEnterで誤動作しない？

### A: CodeMirror 6で対応可能

- `view.composing`フラグでIME変換中を検知可能
- `compositionstart`/`compositionend`イベントで状態管理
- 変換中のEnterキーは無視するロジックを追加

---

## 検証要件

- OS: Windows / Mac / Linux
- ブラウザ: Chrome / Firefox / Safari
- 実施時期: エディタ実装の早期段階でプロトタイプ検証必須
